How to use GetRDone

GetRDone helps you track focused work sessions using the Pomodoro technique — 25-minute blocks of uninterrupted effort.

1

Set up your task types → Setup

Before logging work, create task types that represent categories of work you do — for example: Deep Work, Meetings, Admin, or Learning.

Give each type a unique name and a distinct color. The color appears on charts and log cards to help you spot patterns at a glance.

2

Select a date → Log

On the Log page, use the calendar to navigate to the day you want to log work for. Today is selected by default. Days that already have entries are marked with a dot.

3

Log completed sessions

With the right date selected, click + Add Task Done.

  • Pick the task type from the dropdown.
  • Set the number of 25-minute increments you completed (1–8).
  • Optionally add a brief summary of what you accomplished.
  • Hit Save.

Each entry can be edited (pencil icon) or deleted (trash icon) from the log list.

4

Review your progress → Reports

Pick a date range to see a breakdown of how you spent your time:

  • The stacked bar chart shows daily totals by task type.
  • The line chart shows overall time-on-task trends over the period.

Use this to reflect on your focus habits and adjust your priorities.

Tips

  • Log sessions right after you finish them — it only takes a few seconds.
  • Keep task types broad enough to be reusable, not one-per-project.
  • One increment = 25 minutes. Eight increments = a full 3h 20min focus day.
  • Days with logged entries are highlighted on the calendar with a dot.
